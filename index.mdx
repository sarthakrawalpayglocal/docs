PayGlocal is a global payment gateway that helps merchants accept payments from customers around the world. Whether you're processing domestic transactions or cross-border payments, PayGlocal's API gives you the tools to collect, manage, and reconcile payments with ease.

## Get started in 3 steps

<Steps>
  <Step title="Create your account">
    Sign up for a PayGlocal merchant account and access your dashboard to retrieve your API credentials.
  </Step>
  <Step title="Integrate the API">
    Use your API keys to authenticate requests and initiate your first payment using the PayGlocal REST API.
  </Step>
  <Step title="Go live">
    Test your integration end-to-end in the sandbox environment, then switch to production credentials to start accepting real payments.
  </Step>
</Steps>
