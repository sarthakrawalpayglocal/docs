PayGlocal is a REST API payment gateway that lets you accept domestic and cross-border payments directly from your application. Whether you're building a checkout flow, a mobile payment experience, or an automated billing system, PayGlocal gives you a single API to collect payments via cards, UPI, net banking, and wallets — from customers anywhere in the world.

## What you can build

With PayGlocal, you can:

- **Initiate payments** and redirect customers to a hosted payment page
- **Check payment status** in real time or receive asynchronous updates via webhooks
- **Cancel payments** that haven't been completed
- **Issue refunds** — full or partial — against completed transactions
- **Support multiple payment methods**, including cards, UPI, net banking, and wallets

## Environments

PayGlocal provides two environments so you can develop and test your integration safely before going live.

| Environment | Base URL | Purpose |
|---|---|---|
| Sandbox | `https://api.sandbox.payglocal.in` | Development and testing |
| Production | `https://api.payglocal.in` | Live payments |

<Note>
  Use sandbox credentials exclusively in your sandbox environment. Production API keys will not work against the sandbox base URL, and vice versa.
</Note>

All API requests use HTTPS. Plain HTTP requests are rejected. The API accepts JSON request bodies and returns JSON responses.
